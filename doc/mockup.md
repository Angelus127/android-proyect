<h1>Mockup del proyecto</h1>
	<p> La aplicacion de school pocket estara diseñada para ser agradable a la vista para los estudiantes, se podra observar el mockup interactivo en marvelapp <a href="https://marvelapp.com/prototype/f6hci56/screen/93997556">proyecto completo</a>. </p>
	<p> Por ese motivo la paleta de colores varia segun la funcion que quieras realizar. </p>
		<h2> 1. Menu de inicio </h2>
			<p><blockquote> Cuando el usuario entra en la app podra visualizar todas las funciones que contiene </blockquote></p>

			<p><blockquote> Cuando seleccionamos el boton amarillo, se abrira el apartado de todas las notas que tengas guardadas </blockquote></p>
			<center><img src="image for UI/notes.png" width="280px" height="520px"></center>
		<h2> 3. Visor de notas </h2>
			<p><blockquote> Al seleccionar alguna nota se abrira el visor de lectura, donde podremos ver con mayor clariad la informacion ahi guardada</blockquote></p>
			<center><img src="image for UI/note_view.png" width="280px" height="520px"></center>
		<h2> 4. Calculadora </h2>
			<p><blockquote> Al seleccionar el boton de calculadora, se nos abrira un interfez con una calculadora funcional para resolver problemas matematicos </blockquote></p>
			<center><img src="image for UI/calculator.png" width="280px" height="520px"></center>
		<h2> 5. Diccionario </h2>
			<p><blockquote> Si seleccionamos el boton de diccionario, se desplegara su apartado donde se mostrara un listado de palabras con su significado sacados un API concreta </blockquote></p>
			<center><img src="image for UI/dictionary.png" width="280px" height="520px"></center>
		<h2> 6. Diccionario de idiomas </h2>
			<p><blockquote> Parecido al anterior en este apartado se mostrara un listado de palabras en español y su traduccion en el idioma que desee. La informacion se obtendra de una API concreta  </blockquote></p>
			<center><img src="image for UI/languaje_dictionary.png" width="280px" height="520px"></center>

			<p><blockquote> En este apartado tendremos una tabla periodica interactiva, donde podremos seleccionar el elemento del que queramos conocer una informacion concreta. </blockquote></p>
